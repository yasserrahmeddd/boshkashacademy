# Boshkash Football Academy Management System

A professional, responsive, and robust management system for football academies.

## Features
- **Dashboard**: Real-time statistics.
- **Player Management**: Excel-like table with inline editing, search, and filtering.
- **Finance**: Subscription tracking and invoice generation (Backend ready).
- **Security**: Role-based access and secure authentication.
- **Design**: Modern Dark Message with Glassmorphism.

## Tech Stack
- **Backend**: Python (Flask), SQLAlchemy (SQLite).
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla), Tabulator.js.

## Setup Instructions

1. **Install Python 3.8+**
2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the Application**:
   ```bash
   python app.py
   ```
4. **Access the App**:
   - Open Browser: `http://localhost:5000`
   - **Login**:
     - Username: `admin`
     - Password: `admin122`

## Project Structure
- `app.py`: Main application entry point.
- `models.py`: Database models.
- `routes.py`: API endpoints and views.
- `static/`: CSS, JS, and Assets.
- `templates/`: HTML Templates.
